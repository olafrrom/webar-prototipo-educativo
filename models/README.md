# Carpeta de modelos 3D

Coloca aquí tus modelos en formato `.glb` o `.gltf`.

## Recomendación inicial

Usa archivos `.glb` porque integran geometría, materiales y texturas en un solo archivo.

## Ejemplo de uso en `index.html`

```html
<model-viewer
  src="models/mi-modelo.glb"
  ar
  camera-controls
  auto-rotate>
</model-viewer>
```

## Ideas de modelos educativos

- Molécula de agua
- ADN
- Célula
- Corazón humano
- Cráneo
- Fósil
- Planeta
- Engrane
- Motor simple
